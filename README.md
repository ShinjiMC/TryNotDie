**Try Not to Die** is an immersive Virtual Reality (VR) adventure game built in Unity and designed for Oculus devices.  
Stranded in the depths of space, your ship is in ruins. Your mission? Repair your spacecraft by scavenging the missing pieces scattered across treacherous alien terrains. But bewareâ€”the path is crawling with ferocious space crabs determined to stop you. Can you outsmart them, collect the parts, and escape before it's too late?

## How to Play

1. **Clone the repository locally:**

   ```bash
   git clone https://github.com/ShinjiMC/TryNotDie.git
   ```

2. **Open the project folder in Unity using Unity Hub.**

3. **Connect your Oculus Quest 2 device and hit 'Play' to start the game.**

## Requirements

- Unity (latest stable version)
- Oculus Quest 2 or compatible VR headset
- A USB-C cable (for debugging or initial setup)
